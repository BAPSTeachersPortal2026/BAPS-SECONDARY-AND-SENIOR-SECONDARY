# BAPS Teachers Portal

A web-based Teachers Portal for Bannari Amman Public School.

## Features

- Secondary (Grades IX & X)
- Senior Secondary (Grades XI & XII)
- Lesson Plans
- Assessment Records
- Mark Statements
- Consolidated Mark Statements
- Syllabus Completion Tracking
- Time Tables
- Year Plans
- Special Class Schedules

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages
